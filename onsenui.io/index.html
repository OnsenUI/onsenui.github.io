New & Improved Hybrid Apps

Onsen UI is a front-end development tool that radically improves PhoneGap/Cordova apps. Differing from previous frameworks, it is highly flexible and gives hybrid apps a native look and feel on all platforms. And when used with Monaca, it is simple to add backend features and device support, resulting in a publishable app.

Customizable

A powerful feature of Onsen UI is its customizability. In addition to the themes provided, you can develop your own themes. Each component in a theme can also be customized, by modifying the HTML and CSS code.

Easy To Use

Simply adding a few lines of code will give your application UI components - navigation bars, tab bars, buttons, etc. Since Onsen UI uses AngularJS directives, you can flexibily develop your own original UI components.

  • Navigation Bar

    <ons-navigator title="Page 1"> </ons-navigator>

  • Button

    <ons-button> </ons-button>

  • Switch

    <ons-switch> </ons-switch>

Beautiful & Smooth

Based on the Topcoat CSS framework and with CSS animation, Onsen UI provides a native-like look & feel to PhoneGap apps. Unlike hybrid apps of the past, Onsen UI does not sacrifice user experience for hybrid development.

GUI Tool Coming Soon

With drag & drop support, our GUI tool makes all aspects of front-end development dramatically easier - from page management to UI component layout. Currently under development, it is scheduled for release in early March.

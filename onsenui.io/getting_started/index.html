3.Install nodejs

Download and run the installer from nodejs.org

4.Run the project

Extract the zip file you just downloaded.

Open the terminal (command line) and cd to the extracted folder.

Type: scripts/web-server.js

Then navigate your browser to

http://localhost:8000/app/index.html

5.Start adding your awesome app feature

Your project root is "app" folder. Use your favorite IDE and start adding awesome feature to your app.

You may want to look at "demo" folder for sample codes.
Run the demo code by navigating your browser to:

http://localhost:8000/demo/index.html

6. Build your app:

You can use Cordova to build Onsen UI project

Type: npm install -g cordova

cordova create MyApp && cd MyApp

copy all the files in the app folder of the template to MyApp/www folder

cordova platform add android

cordova build android

cordova emulate android
